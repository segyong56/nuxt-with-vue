# nuxt-with-vue



### 실행방법

- `npm install` , node_modules를 실행해준다.
- `npm run dev` , localhost:3000에서 vue를 실행해준다.

### 디렉토리

**화면을 구성하는 view page**
- pages
  - index.vue
  - profile.vue
  - signup.vue

**공통기능으로 재사용성을 가진 components**
- components
  - CardPost.vue
  - LoginForm.vue

**공통 레이아웃 설정**
- layouts
  - default.vue

**modules 사용설정된 파일**
- nuxt.config.js
